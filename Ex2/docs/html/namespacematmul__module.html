Perform matrix multiplication ( C = A \times B ) using column-by-row order.

This subroutine computes the product of two matrices using the row-by-col algorithm.

Parameters
[in]A(m,n)Left-hand matrix operand.
[in]B(n,p)Right-hand matrix operand.
[out]C(m,p)Result matrix.
[in]mNumber of rows in A.
[in]nNumber of columns in A and rows in B.
[in]pNumber of columns in B.
[in,optional]do_print Logical flag to print the resulting matrix.
Note
Calls 'checkpoint' from the 'debugger' module to ensure the dimension match of the two matrices.
