\doxysubsection{Detailed Description}
Module for handling complex single-\/precision matrices. 

This module defines a custom type \textquotesingle{}complex8\+\_\+matrix\textquotesingle{} and provides several operations such as initialization, random filling, trace computation, adjoint, writing to file, and deallocation.
